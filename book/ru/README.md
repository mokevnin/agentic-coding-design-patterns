# Agentic Design Patterns

Каталог паттернов агентного программирования в духе «Банды четырёх». Каждый
паттерн — это проверенное решение повторяющейся задачи при работе с ИИ-агентом:
как поставить задачу, дать контекст, проверить результат и организовать проект.
Отдельная глава — **анти-паттерны**: типовые ошибки взаимодействия с агентом.

Паттерны сгруппированы по областям (см. [оглавление](SUMMARY.md)). Порядок и
состав каталога заданы в `SUMMARY.md` — он же служит навигацией книги.

Начните с [предисловия](preface.md) и главы [«Как читать книгу»](how-to-read.md).
